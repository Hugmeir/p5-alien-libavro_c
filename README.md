# NAME

Alien::libavro\_c - libavro\_c, with alien

# VERSION

Version 0.03

# SYNOPSIS

    use Alien::libavro_c;

    Alien::libavro_c->libs;
    Alien::libavro_c->libs_static;
    Alien::libavro_c->cflags;

# DESCRIPTION

`Alien::libavro_c` is an `Alien` interface to `libavro-c`.
